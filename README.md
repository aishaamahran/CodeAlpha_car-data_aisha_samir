# Car Price Prediction using Machine Learning

This project predicts car selling prices using Linear Regression and machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data Cleaning
- Label Encoding
- Feature Engineering
- Linear Regression Model
- Model Evaluation
- Car Price Prediction
- Data Visualization
- Correlation Heatmap

## Dataset
Dataset used:
`car data.csv`

## Machine Learning Workflow
1. Load Dataset
2. Data Cleaning
3. Encode Categorical Data
4. Split Training & Testing Data
5. Train Linear Regression Model
6. Evaluate Model Performance
7. Predict Car Prices

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R2 Score

## Visualizations
- Actual vs Predicted Prices
- Feature Correlation Heatmap

## How to Run

```bash
python car_price_prediction.py
